URL Shortener

  • URL Shortener
  • API

API

http://example.com/api.php?parameters

Request Parameters

  • url: (required) The URL to start with http://, https:// or ftp://
  • format: (optional)
    • plain (default) indicates that the output should be Plain Text. Only the URL or an error text is output.
    • json (recommended) indicates that the output should be JSON (JavaScript Object Notation). It will return status(true), longurl, shortid, shorturl or in case of error status(false), longurl und text
  • callback: (optional)

Example PHP

$shortenedurl = file_get_contents('http://myexample.com/api.php?url=http://google.de/');

Example Javascript

$.getJSON('http://myexample.com/api.php?callback=?', {
  url: 'http://google.de/',
  format: 'json'
}, function(res) {
  if(res.status === true) {
    console.log(res.shorturl);
  }
});
